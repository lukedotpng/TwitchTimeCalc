# TwitchTimeCalc - A Chat Bot For Hitman WoA Speedrunners
## Commands
### !time - Hitman WoA Milliseconds Time Calculator
This command is a milliseconds time calcualator for Hitman 3. The calculations are from [Solderq35](https://github.com/solderq35)'s Hitman time calculator website, found [here](https://solderq35.github.io/time-calc-under-5/). 
To use the command, you type `!time`, followed by the score shown on the level rating screen. If a valid time is given, the bot will send the calculated time in format of **_minutes:seconds.milliseconds_** (`00:00.000`). If an invalid score is given, then the message `invalid score :(` is sent. If no score is given, then the message `Time Calc Example: !time 199622` is sent.
